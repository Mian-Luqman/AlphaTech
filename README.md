# AlphaTech Agency Website

A modern, highly interactive, and lightning-fast digital agency website built with Next.js 14. The platform showcases a seamless UI/UX with smooth scroll physics, custom WebGL fluid cursor animations, and stunning glassmorphism aesthetics. Designed to provide digital and utility services with a premium experience.

## ✨ Features

- **Next.js App Router:** Fast navigation, server-side rendering, and advanced routing.
- **WebGL Fluid Simulation:** Custom, high-performance fluid interaction tied to the user's cursor.
- **Smooth Scrolling Physics:** Powered by Lenis for buttery-smooth scrolling experiences.
- **Complex GSAP & Framer Motion Animations:** Scroll-triggered text sliders, staggered grids, and dynamic layout transitions.
- **Glassmorphism UI:** Advanced blurred backdrops, glowing boundaries, and mix-blend visuals.
- **Responsive Design:** Completely optimized for Mobile, Tablet, and Desktop using Tailwind CSS perfectly.

## 🛠️ Technology Stack & Dependencies

To run this project, make sure these core dependencies are installed (they are included in `package.json`):

**Core:**
- `next` (v14.2+)
- `react` & `react-dom` (v18+)
- `typescript` & `@types/node`

**Styling & UI:**
- `tailwindcss` (v3.4+)
- `sass` & `scss` (For global advanced stylesheets)
- `clsx` & `tailwind-merge` (Dynamic class bindings)
- `lucide-react` (Modern SVG Iconography)

**Animations & Physics:**
- `framer-motion` (Declarative component animations)
- `gsap` (Advanced DOM & CSS timeline animations)
- `lenis` (High-performance smooth scrolling engine)

## 🚀 Installation & Setup

1. **Clone the repository:**
   \`\`\`bash
   git clone https://github.com/your-username/agency-website-main.git
   cd agency-website-main
   \`\`\`

2. **Install all required dependencies:**
   \`\`\`bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   \`\`\`

3. **Start the development server:**
   \`\`\`bash
   npm run dev
   \`\`\`
   *Note: Next.js will typically run on `localhost:3000` (or `localhost:3400` depending on the custom startup scripts).*

4. **Build for Production:**
   \`\`\`bash
   npm run build
   npm run start
   \`\`\`
